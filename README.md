# helm-pinyin

Pinyin support for `helm-find-files` and `helm-mini`.

## Installation

``` elisp
(quelpa '(helm-pinyin :repo "twlz0ne/helm-pinyin" :fetcher github))
```

## Usage

``` elisp
(require 'helm-pinyin)
(turn-on-helm-pinyin)
```
